**Orbis** — Это лаунчер для проектов на движке [VoxelCore](https://github.com/MihailRis/VoxelEngine-Cpp), который реализует систему инстантов и групп, где у инстантов в одной группе есть общий контент, который распространяется на все инстанты
---

### О проекте:
- **Язык**: Kotlin
- **GUI фреймворк**: JavaFX
